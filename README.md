🌐 CircleUp — Structured Learning Community Platform

🚀 Learn together. Grow consistently. Build with purpose.

CircleUp is a full-stack web application designed to transform the way people learn by combining community interaction, mentorship, challenges, and personal productivity into one unified platform.
It enables users to join topic-based communities, collaborate with peers, gain recognition through contributions, and progress into mentorship roles while maintaining consistent learning habits.

🔥 Core Features

👤 Authentication & User Management

Secure signup and login using JWT-based authentication
Password encryption using bcrypt
Forgot password and reset password functionality
Profile management (name, description, profile photo)

🎯 Onboarding & Personalization

Topic selection during onboarding
Intelligent circle suggestions based on selected interests
Seamless entry into relevant learning communities

👥 Circles (Learning Communities)

Create and join topic-based circles
View circle details including: 
Members
Posts
Challenges
Leave circles anytime
Mentor-controlled circle management

💬 Posts & Discussions

Create posts within circles
Reply to posts and engage in discussions
Upvote helpful replies
Mark replies as solutions
Reputation system based on: 
Upvotes
Solved responses

🧠 Role Progression System

Automatic promotion system: 
User → Contributor → Mentor
Contributor eligibility based on: 
Solved replies
Upvotes
Circle participation
Mentor eligibility based on: 
Higher contribution metrics
Community validation

🧑‍🏫 Mentorship System

Apply for mentor role
Admin approval workflow
Mentor dashboard to: 
Manage circles
Create challenges
Track performance

🎯 Challenges System

Mentor-created structured challenges
Daily tasks with guided learning
Progress tracking for users
Paid and free challenges supported
Challenge overview and review system

💰 Revenue & Monetization

Mentors can earn through paid challenges
Revenue tracking system for mentors
Payment integration for challenge access

📊 Personal Space (Productivity System)

Create folders and tasks
Add resources (links) to tasks
Track study sessions using logs: 
Date
Start time
End time
Edit and delete tasks/logs
Organized personal learning workflow

🔥 Activity Streak System

Daily streak tracking based on user activity
Encourages consistent learning habits
Automatically updates based on daily engagement

🔔 Notifications System

Real-time notifications for: 
Replies
Upvotes
Role promotions
Mentor approvals

🛠 Admin Panel

Manage users and roles
Approve/reject mentor applications
Manage circles and challenges
Handle complaints and feedback
Monitor platform activity

📞 Support System

Submit complaints and feedback
FAQ section for common queries
Direct contact option (email integration)
